# Sidekick Change Log

The Sidekick Change Log is a Binary Ninja sidebar that displays a list of changes that Sidekick has made through the Analysis Assistant, such as renaming functions, adding comments, and more. Each entry in the log displays the timestamp, a high-level description of the change, the file that was modified, and the number of changes made. Each entry contains the list of detailed changes applied for that entry.

The Sidekick Change Log includes entries for all binaries that have been modified by Sidekick, which can be filtered to display only changes made to the current file.

## Actions

The following actions can be performed in the Sidekick Change Log:

| Action | Description | Steps |
| --- | --- | --- |
| **Expand/Collapse Entry** | Expand or collapse the list of detailed changes applied for an entry | Double-click an entry in the change log |
| **Show Timestamps** | Display timestamps for each change entry | Select `Show Timestamps` from the hamburger menu |
| **Filter by Current File** | Filter the change log to display only changes made to the current file | Select `Filter by Current File` from the hamburger menu |
