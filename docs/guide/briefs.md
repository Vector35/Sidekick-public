# Function Descriptions

Sidekick can automatically generate a brief description for a function that gets recorded as function comment. A function description can be applied to the current function by performing one of the following actions:

- Select `Suggestion Function Comment` from the hamburger menu in the Sidekick Suggestions sidebar, right-click the function comment suggestion, and select `Apply`
- Select `Add Function Comment` from the `Plugins->Sidekick` menu
